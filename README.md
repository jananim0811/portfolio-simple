# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**. This project showcases my basic front-end web development skills and serves as an introduction to my profile.

## Features

* 🏠 Home page with a personal introduction
* 📞 Contact page for communication details
* 🎨 Clean and simple user interface
* 📱 Basic responsive design using CSS
* ⚡ Built with semantic HTML and organized CSS

## Technologies Used

* HTML5
* CSS3

## Project Structure

* **Home Page:** Displays my introduction, skills, and portfolio overview.
* **Contact Page:** Provides contact information and a simple way to connect.

## Purpose

This portfolio was created to practice HTML and CSS fundamentals while building a professional online presence. It represents one of my beginner front-end projects and demonstrates my understanding of webpage structure, styling, and navigation.
